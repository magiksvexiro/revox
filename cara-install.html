Cara Menginstal Vexiro❓

Terakhir Diperbarui 29 Mei 2025

Langkah Awal: Aktifkan USB Debugging dan Debug Nirkabel

Sebelum menggunakan fitur lanjutan Vexiro, kalian perlu mengaktifkan USB Debugging dan Debugging Nirkabel di perangkat Android. Berikut langkah-langkahnya:

  1. Buka Pengaturan (Settings) di perangkat Android kalian.
  2. Scroll ke bawah dan pilih Tentang Ponsel (About Phone).
  3. Ketuk Nomor Bentukan (Build Number) sebanyak 7 kali hingga muncul notifikasi bahwa mode pengembang aktif.
  4. Kembali ke menu utama Pengaturan, lalu masuk ke Opsi Pengembang (Developer Options).
  5. Aktifkan USB Debugging.
  6. Aktifkan Debugging Nirkabel.
  7. Jika muncul konfirmasi, pilih Izinkan.

Langkah Kedua: Aktifkan Brevent Dan Shizuku

  1. Aktivasi Brevent ❗

    Jika kamu belum tahu cara mengaktifkan Brevent, kamu bisa mencari tutorial di YouTube dengan kata kunci:

    cara menggunakan aplikasi Brevent

    Pastikan kamu sudah mengikuti panduan aktivasi hingga status Brevent menunjukkan "Server started"
  2. Aktivasi Shizuku Lewat Brevent

    Setelah Brevent aktif, lanjutkan dengan mengaktifkan Shizuku. Caranya:

    1. Buka aplikasi Shizuku.
    2. Pilih metode aktivasi menggunakan ADB non-root.
    3. Akan muncul perintah ADB seperti ini:

      sh /storage/emulated/0/Android/data/moe.shizuku.privileged.api/start.sh
    4. Salin perintah tersebut.
    5. Buka kembali Brevent lalu tempel perintah tadi di bagian input perintah (jika menggunakan mode Brevent ADB), atau jalankan lewat perintah ADB jika kamu mengaktifkannya lewat Termux.
    Setelah perintah dijalankan, status di aplikasi Shizuku akan berubah menjadi Aktif atau Running.
  3. Install Aplikasi Magiks Vexiro ❗

    Jika Shizuku sudah aktif:

    • Hapus versi lama Magiks Vexiro (jika sebelumnya sudah terinstal).
    • Install versi terbaru dari Magiks Vexiro yang sudah mendukung integrasi dengan Shizuku dan Brevent.
    • Izinkan akses Shizuku ke aplikasi Vexiro.

    Selesai❗
    Sekarang kamu sudah siap menjalankan Magiks Vexiro dengan sistem yang optimal berkat bantuan Brevent dan Shizuku.
